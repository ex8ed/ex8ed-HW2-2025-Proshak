# ex8ed-HW2-2025-Proshak

Text analysis algorithms and BERT fine-tuning model

## remark

Описания шагов, визуализация и описания находятся сразу в обоих блокнотах;

- [HW2-Proshak-I.ipynb](https://github.com/ex8ed/HW2-2025-Proshak/blob/main/HW2-Proshak-I.ipynb) содержит в себе первое задание домашней работы, а именно применение и описание работы методов векторизации текста на текстовом корпусе из трех песен на русском языке.

- [HW2-Proshak-II.ipynb](https://github.com/ex8ed/HW2-2025-Proshak/blob/main/HW2-Proshak-II.ipynb) содержит решение задачи классификации отзывов на IMDB через файнтюнинг BERT: подготовку и обработку данных, а также цикл обучения и валидации получившейся модели.
